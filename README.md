# 🎨 Portafolio Web — Christofer Orduz

Este proyecto es un portafolio personal diseñado en Figma y pensado para mostrar información profesional, proyectos y habilidades de forma clara, moderna y visualmente atractiva.

---

## 📌 Estructura del proyecto

El diseño está dividido en 5 secciones principales:

### 1. Header + Navegación

* Contiene el título del portafolio.
* Barra de navegación con accesos a:

  * Sobre mí
  * Trayectoria
  * Proyectos
  * Habilidades
  * Conocimientos
  * Formulario

👉 Objetivo: permitir una navegación rápida y clara entre secciones.

---

### 2. Sobre mí + Trayectoria

* Breve presentación personal.
* Botón destacado: **“Ver mis proyectos”**.
* Imagen de perfil.
* Línea de separación.
* Sección de trayectoria con indicadores visuales.

👉 Objetivo: introducir al usuario y generar interés en el trabajo.

---

### 3. Proyectos

* Tarjetas de proyectos en formato grid.
* Cada tarjeta incluye:

  * Imagen representativa
  * Título
  * Descripción breve

👉 Objetivo: mostrar trabajos realizados de forma visual y organizada.

---

### 4. Habilidades + Conocimientos

* **Tabla de habilidades** (lado izquierdo).
* **Lista de tecnologías** (lado derecho):

  * Python
  * HTML
  * CSS
  * GitHub

👉 Objetivo: evidenciar capacidades técnicas y herramientas dominadas.

---

### 5. Formulario + Collage

* Formulario de contacto con campos básicos.
* Selección de género (masculino / femenino).
* Botón de envío.
* Collage visual decorativo en el lado derecho.

👉 Objetivo: facilitar el contacto y aportar dinamismo visual.

---

## 🎨 Estilo visual

* Paleta principal en tonos **morados/lilas**.
* Botones redondeados.
* Diseño limpio y moderno.
* Uso de tarjetas (cards) para organización.
* Separación clara por secciones.

---

## 🛠️ Tecnologías sugeridas

Para implementar este diseño:

* HTML5
* CSS3 (Flexbox / Grid)
* JavaScript (opcional para interacción)

---

## 🚀 Posibles mejoras

* Animaciones suaves (hover, scroll).
* Versión responsive (mobile-first).
* Integración con backend para el formulario.
* Añadir enlaces reales a proyectos.
* Modo oscuro / claro.
